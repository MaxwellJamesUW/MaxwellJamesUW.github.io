Ideas and Future Work

If I kept working on this, I would add more extensive style changes to change more elements on a page. At this time, certain websites are less affected by the extension than others because they have complicated style rules. I'd also like to save the default style of a page, so when a user unchecks the 'enable styles' checkbox, the page reverts to its default look.
