Description

For this project, I will build on my MP3 game and add a database to store high scores.

The current MP3 game can be played here:

Hoop Shooter Game

Envisioned Interface

Prototype

The prototype will build on my MP3 game with a Firebase back-end to store user high scores. High scores are displayed at the end of a game. Stretch goal this version: users can input their names on a start screen.

Final Version

The final version will build on the prototype and add polished start and end screens, an option to restart the game, and better visuals and sound effects. New features such as a moving basket game mode and mobile optimization will be added time permitting.

Plan

  1. Create Firebase database for high scores, and test sending scores to it.
  2. Display high scores when the player finishes a game
  3. Add basic start screen where user can input their name and hit 'start'
  4. Milestone 1: Prototype due 3/7
  5. Add polished end and start screens to the app
  6. Add a button to restart the game
  7. Add sound effects, better visuals
  8. Milestone 2: Final Version due 3/17

Unknowns

  • How to restart the game without reloading the window
  • Creating different screens and changing between them

Stretch Goals

  • Optimize for mobile
  • Options to play different game modes such as moving basket or platformer.
